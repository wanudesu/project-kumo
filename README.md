# 🇯🇵 Japanese

## 概要

KUMOは、日韓向けの求人情報を地図ベースで確認できる求人マッチングプラットフォームです。

このリポジトリでは、チーム開発プロジェクトの中で私が担当した実装・画面設計・デザイン調整を中心に整理しています。

## 私が担当したこと

* リクルーター向けページの実装
* Figmaを用いた画面設計
* UI/UXの調整
* 全体的なデザイン方向の提案

## 担当範囲

* メインダッシュボード
* カレンダー
* ミニカレンダー連動の right sidebar
* 求人管理
* 求人登録
* 求人修正
* 会社登録
* 採用担当者マイページ

※ 求職者管理・スカウト機能は担当範囲外です。

## 工夫した点

* どのページでも 일정を確認できるよう、ミニカレンダーを right sidebar と連動して配置
* 登録済みの会社情報に応じて、求人登録時の住所が自動で切り替わるように構成
* 画面ごとの見やすさと流れを意識し、Figmaで先に整理した上で実装

# 🇰🇷 Korean

## 개요

KUMO는 한일 구인구직 정보를 지도 기반으로 확인할 수 있는 매칭 플랫폼입니다.

이 저장소는 팀 프로젝트 전체를 설명하기보다, 그 안에서 제가 담당한 구현・화면 설계・디자인 조정을 중심으로 정리한 개인용 포크 레포입니다.

## 제가 담당한 부분

* 리쿠르터 페이지 구현
* Figma를 활용한 화면 설계
* UI/UX 조정
* 전체적인 디자인 방향 제안

## 담당 범위

* 메인 대시보드
* 캘린더
* 미니캘린더 연동 right sidebar
* 공고 관리
* 공고 등록
* 공고 수정
* 회사 등록
* 구인자 마이페이지

※ 구직자 관리, 스카우트 기능은 담당 범위에서 제외됩니다.

## 신경 쓴 점

* 어느 페이지에서든 일정을 볼 수 있도록 미니캘린더를 right sidebar와 연동해 배치
* 등록된 회사 정보에 따라 공고 등록 시 주소가 자동으로 바뀌도록 구성
* 화면의 흐름과 가독성을 고려해 Figma로 먼저 정리한 뒤 구현
